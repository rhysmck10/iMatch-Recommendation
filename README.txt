-- iMatch is a movie recommendation system --

1 - Run: "pip install -r requirements.txt".

2 - Then run: "python app.py".

3 - Then visit the following on a web browser: "http://127.0.0.1:5000".

4 - Register to create a username and password, and then login.

5 - Begin rating movies to be recommended new ones. (5 movies have to be liked to display recommendations!).
